# Phishing Email Investigation – Excel 4.0 Macro Malware Execution

This project documents a simulated SOC investigation from a LetsDefend lab involving a phishing email with a malicious Excel 4.0 macro attachment.

The investigation confirmed that the attachment executed on the LarsPRD endpoint, generated Command & Control communication, and triggered suspicious regsvr32 activity.

Key skills demonstrated:
- SIEM alert triage
- phishing email analysis
- malicious attachment investigation
- sandbox analysis
- endpoint investigation
- IOC extraction
- MITRE ATT&CK mapping
- incident response recommendations
